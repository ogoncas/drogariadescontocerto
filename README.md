# 💊 Drogaria Desconto Certo

Este é o repositório do site institucional da **Drogaria Desconto Certo**. O projeto foi desenvolvido com foco em performance, otimização para motores de busca e design moderno.

## 🚀 Funcionalidades

* **Dark Mode Automático:** Interface que se adapta dinamicamente à preferência do sistema ou navegador do usuário.
* **Design Responsivo:** Totalmente otimizado para dispositivos móveis, tablets e desktops, incluindo um menu de navegação lateral (hamburguer).
* **Integração WhatsApp:** Botões de conversão direta para pedidos via chat em pontos estratégicos da página.
* **Modais Interativos:** Visualização ampliada de imagens da galeria e player de vídeo institucional integrado sem sair da página.
* **Otimização de SEO:** Uso avançado de metatags, Open Graph (Facebook), Twitter Cards e dados estruturados (Schema.org).
* **Mapa Integrado:** iFrame do Google Maps com efeito visual de inversão de cores/tons de cinza adaptativo.

## 🛠️ Tecnologias Utilizadas

* **Frontend:** HTML5 semântico e Tailwind CSS (via CDN) para estilização utilitária.
* **Comportamento:** JavaScript puro (Vanilla JS) para gerenciar o estado dos modais e do menu móvel.
* **Estilização Adicional:** CSS3 customizado para animações (fade-in, zoom-in) e efeitos de hover.
* **Ícones:** FontAwesome 6.5.1.
* **Tipografia:** Inter (textos gerais) e Poppins (logotipo) via Google Fonts.

## 📂 Estrutura do Projeto

* `index.html`: Página principal (Landing Page) contendo a estrutura de destaques, história, localização e contato.
* `script.js`: Arquivo de lógica responsável pela abertura/fechamento dos modais de mídia e menu responsivo.
* `styles.css`: Estilos complementares que adicionam transições suaves, decorações de título e animações.
* `galeria/`: Diretório de armazenamento das mídias da loja (ex: `1.jpeg`, `2.jpeg`, `3.jpeg`, `4.jpeg`).
* `logo.png`: Logotipo da drogaria para exibição no cabeçalho e rodapé.
* `LICENSE`: Arquivo contendo os termos de direitos autorais.
